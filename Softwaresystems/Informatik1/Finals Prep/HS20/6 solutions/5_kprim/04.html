DE: Welche der folgenden Aussagen bezüglich Exceptions und Error Handling sind korrekt?
EN: Which of the following Statements about Exceptions and Error Handling are correct?
  • False
    DE: Eine aufgerufene Funktion kann gleichzeitig einen Wert mittels return und eine Exception mittels raise zurückgeben.
    EN: A called function can simultaneously return a value using return and an exception using raise.
  • Correct
    DE: Code im finally Block wird auch dann ausgeführt, wenn der dazugehörige Code im try Block keine Exceptions verursacht.
    EN: Code contained in a finally block is executed even if Code contained in the corresponding try block has not caused any exceptions.
  • Correct
    DE: Eine syntaktisch korrekte Anwendung von try/except/else/finally benötigt mindestens try und except oder try und finally.
    EN: Syntactically correct use of try/except/else/finally needs at least try and except or try and finally.
  • Correct
    DE: Es ist möglich, verschiedene Exceptions verschieden zu handhaben, indem man mehrere except blocks für die verschiedenen Exception Types implementiert.
    EN: It's possible to handle different exceptions differently by implementing multiple except blocks for the different exception types.
